Pour essayer Blockly4Thymio, il n'y a rien de plus simple : vérifiez les prérequis,
téléchargez le compilateur Blockly4Thymio et rendez-vous dans la page des exercices
sur  http://www.blockly4thymio.net/les-exercices.html
 
Prérequis
- Avoir une machine sous windows (pour l'instant) et .NET, version 3.0 minimum.
- Avoir téléchargé et installé Aseba (pour avoir les drivers USB du robot Thymio,
  sur votre machine), à l'adresse :
  https://aseba.wdfiles.com/local--files/en:wininstall/aseba-bin-1.4.0-git-b2255c6-win32.exe
- Avoir télécharger l'utilitaire qui servira à transmettre le fichier Blockly4Thymio
  au robot Thymio.
- En option, pour pouvoir utiliser les instructions comme "joue un son", vous devez
  avoir une carte MicroSD avec les fichiers sons présents dans le fichier zip :
  http://blockly4thymio.net/files/telechargements/sons-carte-micro-sd.0.2.zip



Compilateur Blockly4Thymio
--------------------------

Ensuite, vous devez télécharger le compilateur Blockly4Thymio. C'est une archive au format
.zip, décompressez la dans le répertoire de votre choix. Dans ce répertoire, il y a un exécutable
nommé blockly4thymio.xx.xx.exe, c'est le compilateur de Blockly4Thymio.

Pour transmettre le fichier Blockly4Thymio au robot Thymio, vous devez installer ensuite,
un utilitaire développé par David Sherman. Cet utilitaire a été crée, dans le cadre d'un
autre projet autour de Thymio, appelé Scratch/Thymio II. Vous pouvez télécharger cet utilitaire
à l'adresse suivante :
https://github.com/davidjsherman/inirobot-scratch-thymioII/releases/download/v0.6.2-alpha/Scratch2-ThymioII-0.6.2-XP.zip .
Copiez le répertoire \bin contenu dans ce fichier .zip dans le répertoire du compilateur
et renommer le répertoire ainsi copié en \transfert_thymio .



Vous allez vérifier que le compilateur fonctionne correctement.
---------------------------------------------------------------

Allumer Thymio et branchez le par USB à votre ordinateur. Lancer le fichier tester-le-compilateur.b4t
présent dans le répertoire du compilateur. La fenêtre suivante apparait, vous demandant d'associer
un exécutable avec les fichiers .b4t . Choisissez l'option Sélectionner un programme dans la liste
des programmes installés, puis le bouton OK.

Une autre fenêtre se présente pour faire l’association des fichiers .b4t .

Vérifier que la case à cocher Toujours utiliser ce programme pour ouvrir ce type de fichier est cochée.
Cliquez sur le bouton Parcourir… et sélectionnez le fichier blockly4thymio.xx.xx.exe dans le répertoire
du compilateur Blockly4Thymio que vous venez de décompresser. Puis le bouton OK.

Le compilateur Blockly4Thymio va se lancer avec le fichier de test tester-le-compilateur.b4t .
Si le compilateur fonctionne sur votre ordinateur, les LEDs de Thymio devraient s'allumer
successivement en rouge, vert et bleu.

L'installation est terminée !!!



Résumé de l'installation
------------------------

- Télécharger et installer Aseba pour les drivers USB.
- Télécharger le compilateur Blockly4Thymio, au format .zip et le décompresser dans un dossier au choix.
- Télécharger l'utilitaire de transfert vers Thymio au format .zip et décompresser le répertoire \bin
  dans le répertoire du compilateur en le renommant \transfert_thymio .
- Associer le fichier tester-le-compilateur.b4t au compilateur.

Actuellement, l'interface de programmation ne fonctionne pas correctement sous Internet Explorer.
Pour rendre l'expérience utilisateur plus agréable, je vous recommande d'utiliser le navigateur
Firefox. Celui-ci permet de lancer directement le compilateur de Blockly4Thymio lorsque le bouton
"Envoyer à Thymio" est cliqué.


Et maintenant, amusez-vous Thymio et Blockly4Thymio !
